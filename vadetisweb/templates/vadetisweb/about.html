Technology
Vadetis is a web-based client-server application. The client-side is implemented using native JavaScript, jQuery and Highcharts (a JavaScript library for charts and plots), as well as some JavaScript slider libraries (noUiSlider, RoundSlider, and Ion.RangeSlider). The server-side implementation uses Django, a Python web framework, along with several extensions and provides a Django REST Framework API. The data is stored in a MySQL database.
